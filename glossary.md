# Glossary (helpful words)

Short meanings for common words in this guide. Each heading is linkable.

## <a id="apocalyptic-writing"></a>Apocalyptic writing

A symbol-filled writing style about justice and hope.

## <a id="apostle"></a>Apostle

An early Christian messenger; the word means “sent one.”

## <a id="ascension"></a>Ascension

Jesus being taken up to heaven (Christian belief).

## <a id="atonement"></a>Atonement

Making things right after wrongdoing; linked to forgiveness.

## <a id="baptism"></a>Baptism

A Christian water ritual showing commitment and a new start.

## <a id="bible"></a>Bible

Christian sacred writings: many books and types of writing.

## <a id="church"></a>Church

A community of Christians, not just a building.

## <a id="covenant"></a>Covenant

A serious promise-based relationship.

## <a id="crucifixion"></a>Crucifixion

Roman execution on a cross.

## <a id="disciple"></a>Disciple

A close student or follower of a teacher.

## <a id="epistle"></a>Epistle

Another word for a letter.

## <a id="exile"></a>Exile

Being forced to live away from one’s homeland.

## <a id="faith"></a>Faith

Trust; in Christianity, trusting God and Jesus.

## <a id="gentile"></a>Gentile

A non-Jewish person.

## <a id="god"></a>God

The main divine being in Judaism and Christianity.

## <a id="gospel"></a>Gospel

“Good news”; also the four books about Jesus’ life.

## <a id="grace"></a>Grace

God’s unearned kindness (Christian idea).

## <a id="high-priest"></a>High priest

The chief priest in ancient Israel’s worship system.

## <a id="holy-spirit"></a>Holy Spirit

God’s active presence helping and guiding people (Christian belief).

## <a id="idol"></a>Idol

An object or false god treated as most important.

## <a id="idolatry"></a>Idolatry

Worshipping idols.

## <a id="israel"></a>Israel / Israelites

Israel can mean the people, land, or a kingdom; Israelites are the people.

## <a id="jerusalem"></a>Jerusalem

A major city in the Bible’s story.

## <a id="judah"></a>Judah

The southern kingdom in the Old Testament; capital Jerusalem.

## <a id="lord"></a>LORD

Many English Bibles use “LORD” for God’s Hebrew name.

## <a id="messiah"></a>Messiah

“Anointed one”; a chosen leader. Christians apply it to Jesus.

## <a id="parable"></a>Parable

A short teaching story.

## <a id="passover"></a>Passover

A Jewish festival remembering the Exodus from Egypt.

## <a id="pentecost"></a>Pentecost

A Jewish festival; Acts links it to the Holy Spirit.

## <a id="pharisees"></a>Pharisees

A Jewish religious group in Jesus’ time.

## <a id="prayer"></a>Prayer

Speaking to God (silently or aloud).

## <a id="priest"></a>Priest

A religious leader in ancient Israel’s worship system.

## <a id="prophet"></a>Prophet

A person who speaks messages they believe come from God.

## <a id="resurrection"></a>Resurrection

Life after death; Christians believe Jesus rose.

## <a id="sabbath"></a>Sabbath

A weekly rest day in Judaism (also in some Christianity).

## <a id="sacrifice"></a>Sacrifice

An offering given in worship (often animals in the Old Testament).

## <a id="salvation"></a>Salvation

Rescue; in Christianity, being saved from sin.

## <a id="scripture"></a>Scripture

Sacred writings (often the Bible).

## <a id="sin"></a>Sin

Wrongdoing; in Christianity, also a broken relationship with God.

## <a id="synagogue"></a>Synagogue

A Jewish meeting place for prayer and teaching.

## <a id="tabernacle"></a>Tabernacle

A portable worship tent used by the Israelites.

## <a id="temple"></a>Temple

The main worship building in Jerusalem in ancient Israel.

## <a id="torah"></a>Torah

The first five Bible books; also means “instruction” or “law”.

## <a id="worship"></a>Worship

Actions showing devotion to God (prayer, songs, rituals).
